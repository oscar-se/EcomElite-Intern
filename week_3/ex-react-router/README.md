# Ex React Router

Ứng dụng React demo về React Router DOM, hiển thị danh sách người dùng và chi tiết từng user.

| `/` | Home | Trang chủ - Danh sách users |
| `/users/:id` | UserDetail | Chi tiết user theo ID |
| `/about` | About | Trang giới thiệu |
| `/contact` | Contact | Trang liên hệ |

Đây là bài tập thực hành React Router trong chương trình **EcomElite Intern - Week 3**.
