# Todo List App

Ứng dụng quản lý công việc đơn giản được xây dựng bằng **React.js** để ôn lại kiến thức React.

- **Thêm** công việc mới
- **Sửa** công việc đã có
- **Xóa** công việc
- **Lưu trữ** dữ liệu vào LocalStorage

## Kiến thức React đã áp dụng
| `useState` | Quản lý state của todos, editIndex, editText |
| `useEffect` | Load/Save dữ liệu từ LocalStorage |
| `Props` | Truyền dữ liệu giữa các component |
| `Component` | Tách code thành TodoForm, TodoList |
| `Controlled Input` | Input được điều khiển bởi state |
| `Event Handling` | Xử lý sự kiện click, submit, keydown |
